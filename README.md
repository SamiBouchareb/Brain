# 🧠 Brain Training Games

## Overview
A cutting-edge cognitive training web application designed to enhance mental skills through interactive games. Challenge your memory, focus, and reaction time with our engaging mini-games.

## 🎮 Games
- **Memory Match**: Test and improve your memory by matching card pairs
- **Focus Target**: Enhance your reaction time and concentration

## 🚀 Features
- Multiple brain training games
- Real-time score tracking
- Responsive design
- Difficulty levels
- Streak and performance metrics

## 🛠 Technologies
- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion

## 📦 Prerequisites
- Node.js (v14 or later)
- npm (v6 or later)

## 🔧 Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/brain-training-games.git
```

2. Navigate to the project directory
```bash
cd brain-training-games
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```

5. Open `http://localhost:5173` in your browser

## 🎯 How to Play
- **Memory Match**: 
  - Click cards to reveal emojis
  - Match pairs of identical emojis
  - Complete the board with the least number of moves

- **Focus Target**: 
  - Click targets as they appear on screen
  - Build streaks for bonus points
  - Maximize your score before time runs out
